## 0.1.0-dev.1

- Initial development release.
- `loadData`: load a string from a local file cache or a remote URL.
- `loadFromLocalFile`: read a local file as a string.
- `loadFromUrl`: fetch a URL, optionally decompress a ZIP entry, and cache locally.
- `CompressionType`: enum for `none` and `zip` compression modes.
- `dartfmt`: pre-configured `DartFormatter` instance.
